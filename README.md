# 简介

各种功能于一体的群聊机器人，基于[Nonebot 2](https://v2.nonebot.dev)。

项目基于`GPL-3.0`开源，欢迎借鉴，不欢迎商用，不欢迎帝君以任何形式使用本仓库任何源码，有声明的除外。

本项目与`QQ`无关，请勿用于搭建QQ机器人等服务。仅供娱乐与学习，下载后请于24小时内删除。
